porticoAutoload

Automatically load '.class.php' files located in the same directory as portico.class.php porticoAutoload() is then passed to spl_autoload_register() to preserve correct casing; default spl_autoload() forces class names to lowercase before attempting to locate the file
porticoAutoloadFail porticoAutoloadFail will return custom failure instead of E_COMPILE_ERROR on missing porticoAPI child classes
